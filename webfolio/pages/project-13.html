MK Builder

This system manages a construction company, it was built using the following technologies:

Client side: JavaScript, Jquery, JSON, AJAX, HTML5, CSS3 and Bootstrap.

Server side: Slim Framework, PHP, MySql and PDO.

It was built based on SCRUM methodology, REST architectural style and Design Patterns.
